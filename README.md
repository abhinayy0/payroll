<!-- USAGE EXAMPLES -->
## Usage

1. Run the migrations using 
   ```
   python manage.py makemigrations
   ```

2. Apply migrations using 
   ```
   python manage.py migrate
   ```
3. Run the django server using
   ```
   ./start.sh
   ```
